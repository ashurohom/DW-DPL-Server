ERP Labz Tax to GST

ERP Labz Tax to GST is a powerful Odoo module that allows you to convert and rename taxes to GST (Goods and Services Tax) format in Odoo. This module simplifies the process of updating your tax structure to comply with GST requirements, especially for Indian businesses migrating from old tax systems.

Compatible with: Odoo 17.0, 18.0, and 19.0

Key Features

  • Bulk Conversion: Convert all taxes to GST format with a single click
  • Selective Conversion: Choose specific taxes or tax groups to convert
  • Tax Name Updates: Automatically replace "Tax" with "GST" in tax names
  • Description Updates: Update tax descriptions to use GST terminology
  • Tax Group Conversion: Convert tax group names to GST format
  • Preview Mode: Preview changes before applying them
  • Flexible Options: Choose what to update (names, descriptions, groups)
  • Error Handling: Detailed error reporting for failed conversions
  • Safe Operation: Only updates records that contain tax-related terms
  • User-Friendly Interface: Easy-to-use wizard for conversion

What Gets Converted

The module converts the following:

  • Tax Names: "Sales Tax" → "Sales GST", "Purchase Tax" → "Purchase GST"
  • Tax Descriptions: Updates descriptions containing "Tax" or "VAT" to "GST"
  • Tax Group Names: "Tax Group" → "GST Group"

How to Use

  1. Install the module from Apps menu
  2. Go to Accounting → Convert Tax to GST
  3. In the wizard:
    • Choose conversion type: "Convert All Taxes" or "Convert Selected Taxes"
    • If selecting specific taxes, choose the taxes and tax groups to convert
    • Select what to update:
      • Update Tax Names
      • Update Tax Descriptions
      • Convert Tax Groups
      • Update Tax Group Names
  4. Click Preview to see what will be changed (optional)
  5. Click Convert to GST to apply the changes
  6. Review the results message

Conversion Examples

Tax Name Conversion:

  • "Sales Tax 18%" → "Sales GST 18%"
  • "Purchase Tax 5%" → "Purchase GST 5%"
  • "Output Tax" → "Output GST"
  • "Input Tax" → "Input GST"

Tax Description Conversion:

  • "Sales Tax on goods" → "Sales GST on goods"
  • "VAT on services" → "GST on services"

Tax Group Conversion:

  • "Tax Group" → "GST Group"
  • "Sales Tax Group" → "Sales GST Group"

Use Cases

  • GST Migration: Migrate from old tax system to GST
  • Terminology Update: Update tax terminology to GST format
  • Compliance: Ensure compliance with GST regulations
  • Standardization: Standardize tax names across the system
  • Bulk Updates: Update multiple taxes at once
Important Notes:
  • This operation modifies your tax records permanently
  • Always backup your database before running the conversion
  • Use Preview Mode first to see what will be changed
  • The module only updates records containing "Tax" or "VAT" terms
  • Records already containing "GST" will be skipped
